    Bắt đầu với OpenDataMap

    OpenDataMap - Smart Search. Natural Interaction.

    Tổng quan

    OpenDataMap là hệ thống Linked Open Data phục vụ nghiên cứu và chuyển đổi số, bao gồm:

    • Data Pipeline - Thu thập & chuyển đổi dữ liệu OSM → RDF
    • Backend API - NestJS REST API với Fuseki integration
    • Frontend Web - React interactive map application
    • RDF Triplestore - Apache Jena Fuseki server
